# Drug Repurposing AI

An AI-powered machine learning system that predicts which existing FDA-approved drugs can effectively treat new diseases by analyzing gene expression patterns.

## 🎯 Project Overview
- **Accuracy:** 76%
- **Genes Analyzed:** 100
- **Drug Samples:** 500
- **Predictions:** 50 drug-disease combinations

## 📁 Files
- `drug_repurposing_project.py` - Main code
- `drug_repurposing_results.png` - Visualizations
- `drug_repurposing_report.txt` - Detailed report

## 🚀 Quick Start
```bash
python drug_repurposing_project.py
```

## 📊 Key Results
- **Top Prediction:** DRUG_8 + Disease_1 (51% efficacy)
- **Important Genes:** GENE_35, GENE_15, GENE_98
- **Model:** Random Forest Classifier

## 💡 Real-World Impact
- Cost Savings: Reusing existing drugs saves billions in R&D
- Time Savings: Clinical trials already exist for known drugs
- Patient Impact: Faster treatment options for rare diseases
